
Which of the following are examples of node modules?

a. Express

b. Body-parser

c. Socket.io

d. All of the above

Yes, all of the options (Express, Body-parser, and Socket.io) are examples of Node.js modules.

--Express helps in building web applications by providing a framework with useful features.
--Body-parser is a tool used with Express to handle data sent through HTTP requests, especially from forms.
--Socket.io is a library that allows real-time communication between the server and the client, enabling features like live chat or live updates on a webpage.
